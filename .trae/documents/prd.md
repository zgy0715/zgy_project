# DeepAgent 前端 Demo — 产品需求文档 (PRD)

## 1. 产品概述

DeepAgent 是基于大语言模型的多智能体协作开发平台，前端 Demo 需直观展示"AI Agent 团队协作开发软件"的核心体验——用户用自然语言描述需求，Coder/Reviewer/Tester/Deployer 四个 Agent 像真实团队一样协作，思考链实时可见，代码自动生成与审查。

- **目标用户**：开发者、技术面试官、项目评审者
- **核心价值**：让评审者 30 秒内理解"多 Agent 协作"的差异化体验

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 演示用户 | 免登录直接进入 | 浏览全部功能，体验模拟 Agent 交互 |

### 2.2 功能模块

1. **Landing Page**：品牌展示、核心卖点、架构图、CTA 按钮
2. **Dashboard**：项目列表、统计数据、活动流
3. **项目详情 — 工作流编辑器**：可视化 DAG 编排 Agent 执行流程
4. **项目详情 — Agent 对话**：多 Agent 实时对话、思考链可视化、代码输出
5. **项目详情 — 代码编辑器**：代码预览、Diff 对比、文件树

### 2.3 页面详情

| 页面 | 模块 | 功能描述 |
|------|------|----------|
| Landing Page | Hero 区域 | 动态粒子背景 + 品牌标语 + "开始体验"CTA |
| Landing Page | 特性展示 | 6 大核心特性卡片，hover 动效 |
| Landing Page | 架构图 | 交互式四层架构图，点击展开详情 |
| Landing Page | Agent 介绍 | 4 个 Agent 角色卡片，各自能力说明 |
| Dashboard | 统计卡片 | 项目数/Agent 运行次数/代码生成行数/测试通过率 |
| Dashboard | 项目列表 | 项目卡片网格，显示名称/描述/最近活动/Agent 状态 |
| Dashboard | 活动流 | 实时滚动的时间线，展示 Agent 最近操作 |
| 工作流编辑器 | DAG 画布 | ReactFlow 拖拽式 DAG，4 个 Agent 节点 + 条件边 |
| 工作流编辑器 | 工具栏 | 添加节点/运行/暂停/重置/缩放控制 |

## 3. 核心流程

用户打开 Landing Page → 点击"开始体验" → 进入 Dashboard → 选择/创建项目 → 进入项目详情 → 在工作流编辑器中编排 Agent → 切换到 Agent 对话查看实时交互 → 切换到代码编辑器查看生成结果

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：深色科技风 (Dark Mode First)，致敬 VS Code / Cursor 风格
- **主色**：Indigo (#6366f1) — 品牌色，用于 CTA、选中态、活跃指示
- **辅色**：Emerald (#10b981) — 成功/通过，Amber (#f59e0b) — 警告/思考中，Rose (#f43f5e) — 错误/失败
- **背景**：#09090b (近黑) → #18181b (卡片) → #27272a (悬浮) 三级灰阶
- **字体**：JetBrains Mono (代码) + Geist Sans (UI)，现代感 + 技术感
- **按钮**：圆角 8px，品牌色填充 + hover 加亮，ghost 按钮用边框
- **布局**：左侧固定 Sidebar + 顶部 Header + 主内容区，经典 IDE 布局
- **动效**：页面切换淡入、Agent 思考时脉冲动画、代码流式打字效果、卡片 hover 上浮

### 4.2 页面设计概览

| 页面 | 模块 | UI 元素 |
|------|------|---------|
| Landing Page | Hero | 全屏深色背景 + 动态网格线 + 居中大标题 + 渐变品牌色文字 + CTA 按钮 |
| Landing Page | 特性 | 2x3 网格卡片，深色玻璃态，hover 上浮 + 边框发光 |
| Landing Page | 架构 | ASCII 风格架构图，每层可点击高亮 |
| Dashboard | 统计 | 4 列统计卡片，数字大字号 + 图标 + 趋势箭头 |
| Dashboard | 项目 | 3 列卡片网格，每卡含项目名/描述/Agent 状态点/时间戳 |
| 工作流编辑器 | 画布 | 深色画布 + 网格背景，Agent 节点带品牌色边框 + 状态色指示灯 |
| Agent 对话 | 消息 | 左右分栏，Agent 头像 + 名称 + 气泡消息，思考链折叠区 |
| 代码编辑器 | 编辑器 | VS Code 风格深色主题，左侧文件树 + 右侧代码区 |

### 4.3 响应式

- Desktop-first 设计，最小宽度 1024px
- Dashboard 在 1280px+ 显示 3 列卡片，1024-1280 显示 2 列
- 工作流编辑器和代码编辑器不支持移动端（需大屏操作）

### 4.4 Demo 特殊处理

- 所有数据使用 Mock 数据，不依赖后端 API
- Agent 对话使用预设的模拟流式输出（setTimeout 逐字显示）
- 工作流编辑器预设一个完整的 4-Agent DAG 示例
- 代码编辑器预设示例代码文件
- 无需登录，直接进入 Dashboard
